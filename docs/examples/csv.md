# CSV Validation

DataCheck supports CSV files with automatic encoding detection.

---

## Basic Usage

```bash
datacheck validate data.csv --config rules.yaml
```

---

## Complete Example

See the [Quick Start](../getting-started/quick-start.md) guide for a full CSV validation example.

---

## CSV Features

- ✅ Automatic encoding detection (UTF-8, UTF-8-BOM, ISO-8859-1)
- ✅ Header row detection
- ✅ Delimiter auto-detection
- ✅ Large file support (millions of rows)
