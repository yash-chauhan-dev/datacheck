## How to Contribute

1. **Fork the repository**
2. **Create a feature branch**
3. **Make your changes**
4. **Add tests**
5. **Submit a pull request**

See the [full contributing guide](https://github.com/yash-chauhan-dev/datacheck/blob/main/CONTRIBUTING.md) for details.
