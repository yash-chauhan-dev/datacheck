## Latest Release

### [0.1.0] - 2025-01-01

Initial public release of DataCheck!

**Features**:
- CSV, Parquet, SQLite, DuckDB support
- 5 validation rules (not_null, unique, min, max, regex, allowed_values)
- Beautiful terminal output
- JSON output for automation
- CI/CD ready with exit codes
- 180 tests, 92% coverage
